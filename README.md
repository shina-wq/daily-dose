# DailyDose

A full-stack chronic illness management web app that helps users track medications, appointments, and daily health metrics.

## Tech Stack

- **Frontend:** Next.js, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Neon)
- **AI:** Gemini API

## Features

- Medication management
- Appointment tracking
- Health logs & dashboard
- AI-powered health assistant
- Notifications

## Getting Started

```bash
# Install dependencies
cd dailydose-server && npm install
cd ../dailydose-client && npm install

# Add your environment variables
cp .env.example .env

# Run the app
npm run dev
```
